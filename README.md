# Discord Guild Bot

Character management bot with Google Sheets sync.

## Setup

1. Create `.env` from `.env.example`
2. `npm install`
3. `npm run deploy`
4. `npm start`

## Commands

- `/register-user` - Register main character
- `/edit-user` - Manage characters
- `/view-character` - View profiles
- `/admin-guild-manager sync` - Force sync
- `/admin-guild-manager edit-character` - Edit user
- `/admin-guild-manager stats` - View stats
